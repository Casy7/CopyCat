# CopyCat
A simple web app that lets you share files and text between your devices

## Ідея

**Назва проєкту:** CopyCat

<img src="docs/assets/logo.svg" width="100">

**Проблема:** Передача посилань, невеликих текстових фрагментів або файлів між власними пристроями (наприклад, з ПК на смартфон) часто є незручною: користувачам доводиться використовувати "Збережені повідомлення" в месенджерах, відправляти листи на власну пошту або використовувати хмарні сховища, що вимагає авторизації та зайвих кліків -- в юзера банально може не бути доступа до меседжера на робочому ПК чи на новому пристрої.

**Рішення:** CopyCat створює миттєвий тимчасовий "чат із самим собою" у браузері, доступ до якого здійснюється за коротким 8-значним кодом, дозволяючи анонімно та швидко перекидати текст і файли між девайсами зі збереженням сесії.

**Цільовий користувач:** Будь-яка людина, якій потрібно швидко перенести текст, посилання або файл (до 20 МБ) між своїми пристроями чи поділитися ними з колегою поруч, не витрачаючи час на реєстрацію чи логін.

## Технічні вимоги

### Функціональні вимоги

| # | Вимога | Пріоритет |
|---|---|---|
| FR-01 | Система дозволяє користувачу створити або приєднатися до кімнати шляхом введення 8-значного коду без реєстрації. | Must Have |
| FR-02 | Система дозволяє надсилати та відображати текстові повідомлення в UI у режимі реального часу. | Must Have |
| FR-03 | Система дозволяє завантажувати файли (до 20 МБ) у кімнату та вивантажувати їх на іншому девайсі. | Must Have |
| FR-04 | Система зберігає сесійні кукі на боці користувача для відновлення доступу до чату після закриття сторінки. | Must Have |
| FR-05 | Система надає REST API для програмного завантаження та отримання повідомлень/файлів з кімнати. | Must Have |
| FR-06 | Система автоматично видаляє кімнати та всі їхні файли, якщо в них не було активності більше ніж 24 години. | Should Have |


### Нефункціональні вимоги

| NFR | Вимога | Обґрунтування |
|---|---|---|
| Performance | Затримка доставки повідомлення в UI не перевищує 500ms. | Необхідно для UX стандарту відчуття миттєвого «буфера обміну». |
| Security & Capacity | Жорстке обмеження розміру Multipart-запиту: max-file-size 20MB. | Захист сервера від переповнення пам'яті (OOM) та DoS-атак. |
| Storage | Локальне сховище сервера під файли не перевищує 5 GB (старі файли видаляються примусово). | Запобігання падінню сервера через брак дискового простору, оскільки користувачі анонімні. |

## API Design

<br>

### Запит:

```text
POST /api/v1/rooms/{roomCode}/messages
```

**Призначення:** Надіслати нове текстове повідомлення у конкретну кімнату через API.

**Request Body:**
```json
{
  "content": "Some content idk ig"
}
```
<br>

### Успіх:

```text
Response 201 Created
```
```json
{
  "id": "1042567367568536",
  "content": "Some content idk ig",
  "type": "TEXT",
  "createdAt": "2026-06-03T14:30:00Z"
}
```

### Помилки:
```text
Response 400 Bad Request
```
Якщо content порожній або перевищує ліміт символів.

```text
Response 404 Not Found
``` 
Якщо кімнати з таким roomCode не існує або вона була видалена.

<br>
<br>


### Запит:

```text
POST /api/v1/rooms/{roomCode}/files
```

**Призначення:** Завантажити новий файл або зображення у конкретну кімнату (через `multipart/form-data`).

**Request Body:**
```text
Content-Type: multipart/form-data
file: <binary_data>
```
<br>

### Успіх:

```text
Response 201 Created
```
```json
{
  "id": "10425673675687644",
  "content": "my_file.txt",
  "type": "FILE",
  "downloadUrl": "/api/v1/rooms/A1B2C3D4/files/10425673675687644",
  "fileMetadata": {
    "mimeType": "text/plain",
    "sizeBytes": 1048576
  },
  "createdAt": "2026-06-03T14:32:15Z"
}
```

### Помилки:
```text
Response 400 Bad Request
```
Якщо файл не передано, або його розмір перевищує ліміт у 20 МБ.

```text
Response 404 Not Found
``` 
Якщо кімнати з таким roomCode не існує.

<br>
<br>

### Запит:

```text
GET /api/v1/rooms/{roomCode}/content
```

**Призначення:** Отримати історію всіх повідомлень та завантажених файлів у заданій кімнаті.

**Request Body:**
```text
(Порожньо)
```
<br>

### Успіх:

```text
Response 200 OK
```
```json
[
  {
    "id": "1042567367568536",
    "content": "Some content idk ig",
    "type": "TEXT",
    "createdAt": "2026-06-03T14:30:00Z"
  },
  {
    "id": "10425673675687644",
    "content": "my_file.txt",
    "type": "FILE",
    "downloadUrl": "/api/v1/rooms/A1B2C3D4/files/10425673675687644",
    "fileMetadata": {
      "mimeType": "text/plain",
      "sizeBytes": 1048576
    },
    "createdAt": "2026-06-03T14:32:15Z"
  }
]
```

### Помилки:
```text
Response 404 Not Found
``` 
Якщо кімнати з таким roomCode не існує.
